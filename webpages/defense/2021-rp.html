Context

Please use no more than 150 words to describe the context of your artifact to let the jury member better understand where the artifact is from?

The context of doing this project is: 1) I developed an interest in computer graphics in doing a 3D game HugeLand with my peers, and continued learning Computer Graphics from Professor Brian Barsky of University of California, Berkeley; 2) I was frustrated by drawing multiple repeated drafts of a sensor rig I was designing in my solar panel project from different perspectives because of its complexity; and 3) I was doing the new Makerspace design of the new campus.

But doing a 3D modelling is not a great compensation method: It is too costly and rigid for designers, and has a steep learning curves. Existing heavy-weight 3D software packages are usually not suitable for ideation. Therefore, I designed and developed an 3D sketching app that targets a more natural way for prototyping in 3D: users can create a sketch in 3D and view it from multiple perspectives, without having to create multiple 2D sketches.
